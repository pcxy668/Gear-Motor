# 直流减速电机项目

### 1.目录结构
* Common 公共层
* Core 核心层
* Drivers 驱动层             
* Interface 接口层
* Application 应用层
* MDK-ARM MDK工具链
### 2.硬件组成
*  主控芯片 STM32F103C8T6  
*  电机驱动芯片 L298N  
*  电机 JGA25-370-带编码器-12V-282转 21.3减速比
*  0.96寸OLED屏幕
*  RS485通讯模块
*  12V 2800mAh动力锂电池
*  按键模块
### 3.项目说明
* OLED显示当前电机状态及转速：转速范围±180r/min
* 按键控制转速调整和刹车
* 使用PID算法进行转速稳定控制
* 可与STM32网关模块通过Modbus实现远程控制（使用RS485总线）
